# kachick.github.io

<https://kachick.github.io/>

## How to development

```console
> nix-shell
> deno task serve
Listening on http://localhost:4507/
```

Then you can access to <http://localhost:4507/>

There are some tasks for tests and stylelint

```console
> deno task
> deno task check
> deno task stylelint
...
```
