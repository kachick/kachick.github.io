## How to development

```console
> nix-shell
> deno install --allow-net --allow-read https://deno.land/std@0.159.0/http/file_server.ts
> ~/.deno/bin/file_server public
Listening on http://localhost:4507/
```

Then you can access to http://localhost:4507/
